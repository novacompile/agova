# agova

Robust agent swarm and orchestrator, for advanced prompting and tasks.

---

## Quick Start

For installation on Windows using batch, follow the steps on the [full installation documentation](install/README.md).

```bash
# Clone the repository
git clone https://github.com/yourusername/agova.git
cd agova

# Run the installer
chmod +x install/install.sh
./install/install.sh

# Restart your terminal or run:
source ~/.bashrc  # or source ~/.zshrc

# Start Agova
agova
```
